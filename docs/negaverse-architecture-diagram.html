negaverse — negative-dataset generation pipeline

Universal engine, PPI-anchored MVP. Six layers turn a positive interaction graph into a matched, confidence-scored negative set — scored by three independent streams and split for transductive or inductive learning. Grounded in Park&Marcotte (2011), Koyama et al. (2023), Negatome 2.0 (2014).

Structured stream (rules / DBs) Literature stream (LLM reasoning) Embedding / graph stream Gold negatives (Negatome) Dual output (train / eval)

Two dashed feedbacks encode the paper insights: the embedding stream guides candidate sampling (Layer 1) toward where hard negatives live, and the fused confidence drives a self-training loop (Koyama) that re-scores the pool, promotes near-boundary negatives, and flags high-scoring "negatives" as suspected false negatives. Splitting into a biased/hard train set and an unbiased/representative eval set is Park&Marcotte's non-negotiable — produced together, never merged. Every pair keeps per-stream sub-scores and a provenance trail, so datasets can be re-scored as biology is revised.
